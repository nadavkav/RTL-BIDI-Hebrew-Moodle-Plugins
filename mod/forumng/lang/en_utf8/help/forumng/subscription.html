Subscription

Four subscription options are available for forums:

  • Everyone can choose to be subscribed - everyone who can view discussions in the forum is allowed to subscribe to it if they choose.
  • Force everyone to be subscribed - all users with the relevant subscribe roles are automatically subscribed to the forum and cannot unsubscribe.
  • Everyone is initially subscribed - users in subscribe roles are automatically added to the forum when they receive those roles, but may unsubscribe later if they choose.
  • Subscription is not permitted - email subscription is unavailable for this forum.

The subscribe roles list is a system option. Ordinarily this would include roles which are a core part of a course (such as students) but not roles which are used for management purposes (such as administrators).

Subscriptions stop automatically if the user is no longer allowed to view discussions in the forum (for example, if they are no longer a member of the course).

This option can be disabled at site level in the system settings. If the option is fixed in system settings, then all forums receive the same setting. If the option is not fixed, then it can be changed on each individual forum.
