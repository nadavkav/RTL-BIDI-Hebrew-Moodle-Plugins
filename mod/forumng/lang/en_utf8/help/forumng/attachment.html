Attachments

You can attach files along with your forum post.

  • Any type of file can be uploaded.
  • You can have any number of files in a forum post, but if you want more than three it's often better to put the files together into a zip file before you upload.
  • There is a maximum size limit per file.
  • If you're uploading several files or large files, it make take some time to transfer them to the server when you save your post.
