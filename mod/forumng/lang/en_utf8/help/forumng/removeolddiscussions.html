Remove old discussions after how many months

Choose how long you want to keep the discussions in this forum.

The default option is Never remove. If you choose this option, discussions will never be automatically removed.

Alternatively, you can specify how long you want to keep discussions in this forum. Discussions will be automatically removed if they have had no new posts for a certain amount of time. For example, if you select 6, discussions which have been inactive for 6 months or more will be removed.
