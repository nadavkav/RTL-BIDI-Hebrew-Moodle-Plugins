Paramètre : Autoriser la réduction de la source

Cette option, si elle est active, autorise les participants à effectuer réellement le filtrage des entrées en supprimant de la base de données les informations rejettées. Ceci peut être utile dans un processus collectif ou beaucoup d'idées ont été apportées.

Attention, cette réduction détruira les données enregistrées pour les entrées supprimées pour tous les autres opérateurs. Ceci peut avoir des conséquences sur le sens des rangements déjà effectués.
