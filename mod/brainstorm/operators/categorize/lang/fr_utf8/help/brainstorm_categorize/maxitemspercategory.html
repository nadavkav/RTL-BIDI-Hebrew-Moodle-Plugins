Paramètre : Population maximum d'une catégorie

Par défaut, il n'existe pas de limite dans une catégorie. Toutes les idées pourraient être associées à une seul catégorie proposée. Pour forcer la répartition, vous pouvez imposer un maximum d'associations aux catégories.

Ce réglage n'est actif que dans le cas d'une sélection unique.
